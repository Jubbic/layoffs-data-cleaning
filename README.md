# 📊 SQL Data Cleaning Project – Layoffs Dataset

## Overview

This project demonstrates my SQL data cleaning skills using a real-world layoffs dataset. The objective was to clean, standardize, and prepare the data for exploratory analysis and business reporting.

---

## Project Objectives

- Remove duplicate records
- Standardize inconsistent data
- Handle missing (NULL) values
- Remove unnecessary columns
- Prepare the dataset for analysis

---

## Tools Used

- MySQL
- MySQL Workbench
- GitHub

---

## Dataset

The dataset contains information about company layoffs, including:

- Company
- Industry
- Location
- Country
- Total Laid Off
- Percentage Laid Off
- Date
- Funding Raised
- Company Stage

---

## Data Cleaning Process

### 1. Created a Staging Table
Created a duplicate of the original dataset to preserve the raw data before making any changes.

### 2. Removed Duplicate Records
Used SQL window functions to identify and remove duplicate rows.

### 3. Standardized Data
- Cleaned company names
- Standardized country names
- Corrected inconsistent industry values
- Formatted date values

### 4. Handled Missing Values
- Identified NULL values
- Updated missing values where appropriate
- Removed unnecessary records

### 5. Removed Unnecessary Columns
Dropped temporary columns used during the cleaning process.
